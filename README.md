# spectre-matrix-file
Loader for Spectre VR .MAT files

Supports
- wrld (maps)
- enem (enemy sets)
- wave (levels)
- shap (models)
- mshp (compound models)
- rshp (model variations)

No support yet for
 - bval (enemy data)
 - str#
 - vers
 - sstf (???, as well as extra life score increments)
 - shps (maps objects to rendered models)
 - clut
 - wval
